This project transforms CAM files generated from Onshape CAM Studio into output formatted for a CAMMaster Stinger 2 router.

## Browser Version (GitHub Pages)

The `docs/` folder contains a full client-side web app that lets you:

1. Upload an `.NC` file
2. Run the formatter using Python in the browser (Pyodide)
3. Download the generated `_formatted.tap` file

No backend is required.
